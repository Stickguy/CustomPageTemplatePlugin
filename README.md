# CustomPageTemplatePlugin
Add A Custom Page Template with Custom Metaboxes via a WordPress Plugin

This is a boilerplate plugin to be used to add a custom page template with custom metaboxes to a WordPress Site via a plugin. It's useful for situations where you don't have a child theme available, and you donâ€™t want run the risk of having your changes overwritten when your parent theme gets updated.

- Download or Clone Repo.

- On line 81 of custompagetemplate.php add the template(s) you want to use to the array:
